# retail-data-sanitization-suite
Practical Python toolkit for retail &amp; e-commerce data sanitization. Extracts, cleans, and structures raw web scraping dumps (Farmatodo, EPA, MercadoLibre, FB Marketplace, GMaps) into clean Excel reports. Reliably built with Pandas, Tkinter, and AI-assisted vibecoding for resilient, async-first business automation and robust data pipelines &amp; ops.
